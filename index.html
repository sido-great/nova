About this connection

This app uses OAuth2 Authorization Code + PKCE as its primary login method, with Personal Access Token (PAT) as an alternative. Either way, your credentials stay on Deriv's domain — this app only ever receives tokens. All tokens live only in this browser tab's memory and are cleared on logout or tab close.

Trading is performed over a live WebSocket connection directly to wss://ws.derivws.com — no data passes through any intermediate server we control.

Before you start

This app lets you place real trades on your own Deriv account. Synthetic indices, options and accumulators can lose your full stake quickly, and no chart, statistic or "signal" in this app can predict their outcome — several of these instruments are explicitly generated by a random number process. Only trade money you can afford to lose, and consider starting on a demo account.
